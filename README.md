# Beast Console

## AI Helper Setup

1. Replace `PASTE_MY_OPENAI_API_KEY_HERE` in `.env.local` with a real OpenAI API key.
2. Restart the development server after editing `.env.local`.
3. Run `npm run dev`.
4. Open the dashboard and launch the AI Helper.
5. Send a test message and confirm an assistant reply appears.

`OPENAI_API_KEY` is server-only. Do not rename it with a `NEXT_PUBLIC_` prefix or commit `.env.local`.
